4. **db.Collection_name.find(selection_criteria, projection,options)**

```
- selection_criteria: specifies the selection criteria using query operators. ex: {title:"spaderman"}.
- projection: specifies the fields to return. ex: {title:1,author:1}.
- options: specifies the options for the query. ex: {limit:5}.
ex: db.books.find({title:"spaderman"},{title:1,author:1},{limit:5})
```

5. **db.<collection_name>.insertOne({key:value})**

``` this command will insert a new document in the specified collection. ex:
db.books.insertOne({title:"spaderman",author:"hassane",pages:500,"genres":["love","action"],rating:8})
```

6. **db.<collection_name>.insetMany([{key:value},{key:value}])**

``` this command will insert multiple documents in the specified collection. ex:
db.books.insertMany([{title:"spaderman",author:"hassane  ",pages:500,"genres":["love","action"],rating:8},
{title:"spaderman",author:"hassane ",pages:500,"genres":["love","action"],rating:8}])
```

7. **db.<collection_name>.updateOne({key:value},{key:value})**

``` this command will update the first document that match the specified query. ex:
db.books.updateOne({title:"spaderman"},{$set:{rating:9}}) :
    this will update the rating of the first document that has the title "spaderman" to 9.

db.books.updateOne({title:"spaderman"},{$push:{genres:{ $each: ["comedy","drama"]}})
    this will add the genres "comedy" and "drama" to the first document that has the title "spaderman".

db.books.updateOne({title:"spaderman"},{$push:{genres:"comedy"}) 
    this will add the genre "comedy" to the first document that has the title "spaderman".

db.books.updateOne({title:"spaderman"},{$pull:{genres:"comedy"})
    this will remove the genre "comedy" from the first document that has the title "spaderman".

db.books.updateOne({title:"spaderman"},{$inc:{rating:1}})
    this will increment the rating of the first document that has the title "spaderman" by 1.

```


8. **db.<collection_name>.updateMany({key:value},{key:value})**

``` this command will update all the documents that match the specified query. ex:  
db.books.updateMany({title:"spaderman"},{$set:{rating:9}})
```

9 - Operators and complexe queries:

```

$eq: Matches values that are equal to a specified value.
$gt: Matches values that are greater than a specified value.
$gte: Matches values that are greater than or equal to a specified value.
$lt: Matches values that are less than a specified value.
$lte: Matches values that are less than or equal to a specified value.
$ne: Matches all values that are not equal to a specified value.
$in: Matches any of the values specified in an array.
$nin: Matches none of the values specified in an array.
$and: Joins query clauses with a logical AND returns all documents that match the conditions of both clauses.
$or: Joins query clauses with a logical OR returns all documents that match the conditions of either clause.
$not: Inverts the effect of a query expression and returns documents that do not match the query expression.
$exists: Matches documents that have the specified field.
$type: Selects documents if a field is of the specified type.
$expr: Allows use of aggregation expressions within the query language.
$all: Matches arrays that contain all elements specified in the query.

- ex: db.books.find({$and:[{rating:{$gt:7}},{rating:{$lt:9}}]})
- ex: db.books.find({$or:[{rating:{$gt:7}},{rating:{$lt:9}}]})
- ex: db.books.find({rating:{$exists:true}})
- ex: db.books.find({rating:{$type:"number"}})
- ex: db.books.find({$expr:{$gt:["$rating",7]}})
- ex: db.books.find({genres:{$all:["love","action"]}})

```

10. **db.<collection_name>.deleteOne({key:value})**

``` this command will delete the first document that match the specified query. ex:
db.books.deleteOne({title:"spaderman"})
```

11. **db.<collection_name>.deleteMany({key:value})**

``` this command will delete all the documents that match the specified query. ex:
db.books.deleteMany({title:"spaderman"})
```

12. **db.<collection_name>.drop()**

``` this command will delete the specified collection. ex:
db.books.drop()
```

13. **db.dropDatabase()**

``` this command will delete the current database. ex:   
db.dropDatabase()
```

14. **db.<collection_name>.find().sort({key:1})**

``` this command will sort the documents in the specified collection in ascending order. ex:
db.books.find().sort({rating:1})
```

15. **db.<collection_name>.find().sort({key:-1})**

``` this command will sort the documents in the specified collection in descending order. ex:
db.books.find().sort({rating:-1})
```

16. **db.<collection_name>.find().limit(number)**
``` this command will limit the number of documents returned by the query. ex:
db.books.find().limit(5)
```

17. **db.<collection_name>.find().skip(number)**

``` this command will skip the specified number of documents before returning the results. ex:
db.books.find().skip(5)
```

18. **db.<collection_name>.find().count()**

``` this command will return the number of documents in the specified collection. ex:
db.books.find().count()
```

19. **db.<collection_name>.aggregate([{$match:{key:value}},{$group:{_id:"$key",total:{$sum:1}}},{$sort:{total:-1}}])**

``` this command will perform an aggregation operation on the specified collection. ex:
db.books.aggregate([{$match:{rating:{$gt:7}}},{$group:{_id:"$author",total:{$sum:1}}},{$sort:{total:-1}}])
```
