## How To Connect Python With MongoDb

1. Install pymongo package

```bash
pip install pymongo
```

2. Import pymongo package

```python
import pymongo
```

3. Connect to the MongoDB server

```python
client = pymongo.MongoClient("mongodb://localhost:27017/")
```

4. Create a database

```python
db = client["bookstore"]
```

5. Create a collection

```python
collection = db["books"]
```

6. Insert a document

```python
book = {"title": "spaderman", "author": "hassane", "pages": 500, "genres": ["love", "action"], "rating": 8}
collection.insert_one(book)
```
