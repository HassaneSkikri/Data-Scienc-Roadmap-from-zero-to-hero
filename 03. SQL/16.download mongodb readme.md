## Instructions

1. [Download the Windows installer](https://www.mongodb.com/try/download/community)
2. Step through installer (be sure to download compass)
3. Add mongod.exe to path
4. Create `/data/db` directory
5. Create this at root directory of hard-drive
6. Ensure that current user has read/write permissions
7. Or customize location using: `mongod –dbpath`I put this at the root directory of my home folder
8. Start MongoDB server
        
        mongod
9. Connect to MongoDB server from client – (in new terminal window) mongo –host localhost:27017 – (in compass) use the default settings
